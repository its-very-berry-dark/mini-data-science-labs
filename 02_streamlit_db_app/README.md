### Python and SQL Database Implementation using Streamlit
This laboratory activity was created in my university course and employs the use of Python to create and manipulate a SQLite database and integrate it into a Streamlit webapp. CRUD operations will be performed through a web interface. Certain constraints are set for this activity to improve data collection in the database, such as email formatting (including email symbols), age range, and name constraints (no numbers).

Link for Streamlit Cloud: https://mini-data-science-labs-bg9tvxv2uaxsfznjycdmwh.streamlit.app/
